# Meu Projeto Fullstack

Projeto contendo Frontend e Backend.

## Estrutura

meu-projeto/

├─ frontend/ # Código do front-end

├─ backend/ # Código do back-end

└─ README.md

## Rodando o projeto localmente

### Backend

1. Entre na pasta do backend:

```bash
cd backend
```

# Instale dependências:

```bash

# Node.js

npm install

# ou

yarn install

# Python

pip install -r requirements.txt
```

# Rode o servidor:

```bash
# Node.js

npm start

# Python

python app.py # ou comando do framework usado
```

## Frontend

Entre na pasta do frontend:

```bash
cd frontend

Instale dependências:

npm install

# ou

yarn install
```

## Rode o servidor de desenvolvimento:

```bash
npm start
# ou
yarn start
```
